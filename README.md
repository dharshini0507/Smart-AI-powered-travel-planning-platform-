# Smart AI Powered Travel Planning Platform

## Overview

Smart AI Powered Travel Planning Platform is an intelligent travel recommendation system that helps users discover destinations, generate personalized itineraries, and optimize travel plans based on preferences and budget constraints.

## Features

* Personalized Travel Recommendations
* AI-Based Destination Suggestions
* Budget Optimization
* Dynamic Itinerary Generation
* User Preference Analysis
* REST API Integration
* Responsive User Interface

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Flask
* Python

### AI / Machine Learning

* Recommendation Engine
* Preference Analysis
* Collaborative Filtering

## Project Workflow

1. User enters travel preferences.
2. AI analyzes destination interests.
3. Recommendation engine suggests locations.
4. Budget optimizer ranks available options.
5. Personalized itinerary is generated.

## Installation

### Clone Repository

```bash
git clone https://github.com/dharshini0507/Smart-AI-powered-travel-planning-platform-.git
```

### Install Dependencies

```bash
npm install
pip install -r requirements.txt
```

### Run Backend

```bash
python app.py
```

### Run Server

```bash
node server.js
```
